### AGENTS-HUB (`~/Projects/agents-hub`)

Live at `https://agents-hub-fawn.vercel.app`. Production-grade SEO engine. Auth: `x-api-key` (public, rate-limited) + optional `x-internal-key` (server-to-server).

API key model (`AGENTS-HUB src/lib/api-key.ts`): SHA-256 hashed in Prisma `apiKey` table. Two scopes:
- `public` — sent as `x-api-key`, rate-limited per key
- `internal` — sent as `x-internal-key`, bypasses public limits, server-only

**Generation script:** `~/Projects/agents-hub/scripts/generate-api-key.ts`. Usage:
```bash
pnpm tsx scripts/generate-api-key.ts --label="<label>" --scope="public|internal" --rateLimit=N
```
The plaintext key prints ONCE. Hash goes to DB.

CrawlIQ-side proxy client: `src/lib/agents-hub-client.ts`. Activates when these env vars are set on CrawlIQ Vercel:
- `AGENTS_HUB_URL` = `https://agents-hub-fawn.vercel.app`
- `AGENTS_HUB_PUBLIC_KEY` = `chk_pub_<...>`
- `AGENTS_HUB_INTERNAL_KEY` = `chk_int_<...>`

**Until these env vars are set, CrawlIQ runs its 1-shot Cheerio + Groq local fallback — i.e. ~10% of its real engine.** Generating the keys + wiring them is the highest-priority unblock pending user authorization.

## 17. Definition of done

A task is done only when:

1. The change is in a commit (not just edited file)
2. `pnpm exec tsc --noEmit` exits 0
3. The deployed preview URL shows the change in a real browser (verify via Playwright screenshot at desktop + mobile viewports — DO NOT trust the diff alone)
4. Console errors captured by Playwright are zero (or explicitly understood + accepted)
5. If user-visible, the new preview URL is posted with a 1-line summary

A cluster is done only when all its tasks are done AND committed AND deployed AND screenshot-verified.
